# Note.com ProseMirror Markdown サポート調査

## 調査日: 2025-12-05

## テスト方法

すべてのMarkdown記法をClipboardEvent経由でProseMirrorエディタにペーストして確認。

## 結果

### ✅ サポートされている

| Markdown記法 | 記述例 | 結果 |
|-------------|-------|------|
| 見出し | `# H1`, `## H2`, `### H3` | ✅ h2, h2, h3に変換 |
| 太字 | `**text**` | ✅ `<strong>` |
| 取り消し線 | `~~text~~` | ✅ 取り消し線表示 |
| コードブロック | ` ``` `code` ``` ` | ✅ 黒背景のコードブロック |
| リンク | `[text](url)` | ✅ `<a>` |
| 引用 | `> text` | ✅ `<blockquote>` グレー背景 |
| 箇条書き | `- item` | ✅ `<ul><li>` |
| 番号付きリスト | `1. item` | ✅ `<ol><li>` |
| 水平線 | `---` | ✅ `<hr>` (確認済み) |

### ❌ サポートされていない

| Markdown記法 | 記述例 | 結果 |
|-------------|-------|------|
| 斜体 | `*text*`, `_text_` | ❌ 通常テキスト |
| インラインコード | `` `code` `` | ❌ バックティックのまま表示 |
| テーブル | `\| A \| B \|` | ❌ Markdown記法のまま表示 |
| 画像 | `![alt](url)` | ❌ テキストのまま |

## 重要な発見

1. **見出しレベル**: `#` (H1) は `<h2>` に変換される
2. **コードブロック**: 認識されるが、インラインコードは認識されない
3. **画像**: Markdown記法は完全に非サポート

## 実装への影響

- Markdownで記事を書く場合、上記サポート範囲を考慮
- 画像は別途アップロード処理が必要（Markdown記法では挿入不可）
- インラインコードは使用できない（代替策検討が必要）
