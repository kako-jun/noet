# スクリーンショット予定地

このディレクトリには以下のスクリーンショットを配置する予定です：

1. `login.png` - 認証プロセスのスクリーンショット
2. `new-article.png` - 記事作成のスクリーンショット
3. `publish.png` - 記事投稿のスクリーンショット
4. `list.png` - 記事一覧表示のスクリーンショット
5. `demo.gif` - 使用デモのGIFアニメーション（オプション）

## スクリーンショット取得方法

```bash
# Asciinemaでターミナルセッションを記録
asciinema rec demo.cast

# terminalizer で記録
terminalizer record demo

# GIFに変換
terminalizer render demo
```
